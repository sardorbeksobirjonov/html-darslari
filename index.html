📚 HTML Darslari

HTML nima?

HTML — HyperText Markup Language degani bo‘lib, bu veb-sahifalarni tuzish tili.

📌 Nima uchun ishlatiladi?
• Saytning tuzilmasini yaratish (matn, rasm, video, tugma joylash).
• Brauzerga nimani va qayerda ko‘rsatish kerakligini aytadi.

💡 Oddiy qilib aytganda:
HTML – saytning skeleti 🦴,
CSS – uning ko‘rinishi 🎨,
JavaScript esa harakati ⚙️.

<!DOCTYPE html> — hujjat turini bildiradi (HTML5).

<html> — butun HTML hujjatni o‘rash uchun.

<head> — sahifa haqida ma’lumot (title, meta, style).

<title> — brauzer oynasida ko‘rinadigan nom.

<body> — foydalanuvchiga ko‘rinadigan barcha kontent shu yerda.

<!DOCTYPE html>
<html>
  <head>
    <title>Sahifa nomi</title>
  </head>
  <body>
    Kontent shu yerda
  </body>
</html>

📌 HTML — <nav> Tegidan Foydalanish

<nav> — sahifa yoki sayt ichida navigatsiya (menyu) yaratish uchun ishlatiladi.

<ul> — tartibsiz ro‘yxat (bullets bilan).

<li> — ro‘yxatning bitta elementi.

<a> — havola (link) yaratadi, href atributi orqali qayerga o‘tishini ko‘rsatadi.

Odatda, <nav> ichida <ul>, <li> va <a> birgalikda ishlatiladi.

  • Bosh sahifa
  • Biz haqimizda
  • Xizmatlar
  • Aloqa
<nav>
  <ul>
    <li><a href="#">Bosh sahifa</a></li>
    <li><a href="#">Biz haqimizda</a></li>
    <li><a href="#">Xizmatlar</a></li>
    <li><a href="#">Aloqa</a></li>
  </ul>
</nav>

📌 HTML — Ro‘yxat Teglari (<ul>, <li>, <ol>)

<ul> — Tartibsiz ro‘yxat yaratadi (belgilari nuqta yoki boshqa markerlar bo‘lishi mumkin).

<li> — Ro‘yxatning har bir elementi (list item).

<ol> — Tartibli ro‘yxat yaratadi (raqamlar bilan).

📌 Eslatma: <li> har doim <ul> yoki <ol> ichida ishlatiladi.

Tartibsiz Ro‘yxat (ul)

  • Olma
  • Banan
  • Uzum

Tartibli Ro‘yxat (ol)

  1. HTML o‘rganish
  2. CSS o‘rganish
  3. JavaScript o‘rganish
<!-- Tartibsiz ro‘yxat -->
<ul>
  <li>Olma</li>
  <li>Banan</li>
  <li>Uzum</li>
</ul>

<!-- Tartibli ro‘yxat -->
<ol>
  <li>HTML o‘rganish</li>
  <li>CSS o‘rganish</li>
  <li>JavaScript o‘rganish</li>
</ol>

📱 HTML & CSS — Responsiv Dizayn Asoslari

1️⃣ <meta name="viewport"> — brauzerga sahifani qurilma o‘lchamiga moslashtirishni aytadi.

2️⃣ Moslashuvchan o‘lchamlar — px o‘rniga %, em, rem, vw, vh ishlatish.

3️⃣ @media queries — ekran o‘lchamiga qarab alohida CSS yozish.

Bu usul bilan sayt telefon, planshet va kompyuterda yaxshi ko‘rinadi.

<!-- 1. Viewport meta -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<style>
/* 2. Moslashuvchan o‘lcham */
img {
  width: 100%;
  max-width: 400px;
}

/* 3. Media queries */
@media (max-width: 600px) {
  body { background: lightyellow; }
}

@media (min-width: 601px) and (max-width: 1024px) {
  body { background: lightblue; }
}

@media (min-width: 1025px) {
  body { background: white; }
}
</style>
