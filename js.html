JavaScript — tushuntirish, kod va natija Live

JavaScript nima?

JavaScript — bu veb-sahifalarga interaktivlik qo‘shish uchun ishlatiladigan dasturlash tili. HTML va CSS faqat tuzilma va dizayn yaratadi, JavaScript esa unga jon bag‘ishlaydi.

Nega ishlatiladi?

  • Veb-sahifalarni dinamik qilish
  • Formalarni tekshirish
  • Animatsiyalar yaratish
  • O‘yinlar va interaktiv dasturlar tuzish

Kod namuna

<script>
document.getElementById("natija").innerText = "Salom, JavaScript ishladi!";
</script>
JavaScript natijasi bu yerda chiqadi

O‘zgaruvchilar (Variables)

JavaScript’da ma’lumotlarni saqlash uchun var, let va const ishlatiladi.

  • var — eski usul, barcha joyda ishlaydi
  • let — o‘zgaradigan qiymat uchun
  • const — o‘zgarmas qiymat uchun
let ism = "Sardor";
const yosh = 20;
document.getElementById("varNatija").innerText =
  `Mening ismim ${ism}, yoshim ${yosh} yosh.`;
Natija bu yerda chiqadi

Funksiya

Funksiya — bu kodni qayta-qayta ishlatish uchun yaratilgan blok.

Uni function kalit so‘zi bilan yaratamiz. 

function salomBer(ism) {
  return `Salom, ${ism}!`;
}
document.getElementById("funcNatija").innerText = salomBer("Sardor");
Natija bu yerda chiqadi

JavaScript Funksiyalar

Funksiya nima?

Funksiya — bu ma’lum vazifani bajaradigan kod bo‘lagi. Uni chaqirsak, ichidagi kod ishlaydi.

Afzalliklari:

  • Kodni qayta-qayta yozmaslikka yordam beradi
  • O‘qilishi va tushunilishi oson
  • Bitta joydan boshqariladi

Kod misoli

function salomBer(ism) {
  return "Salom, " + ism + "!";
}

document.getElementById("natija").innerText =
  salomBer("Sardor");
Natija bu yerda chiqadi
