🎨 CSS — Eng Kerakli Property’lar Qo‘llanmasi

1️⃣ Matn va Shrift

color — matn rangini belgilaydi.

font-size — matn shrift o‘lchamini belgilaydi.

font-family — shrift turini belgilaydi.

font-weight — matn qalinligini belgilaydi.

text-align — matn joylashuvini belgilaydi.

.text {
  color: red;
  font-size: 20px;
  font-family: Arial;
  font-weight: bold;
  text-align: center;
}

Bu matn qizil, 20px, Arial shriftida va markazda.

2️⃣ O‘lcham va Bo‘shliq

width — kenglik.

height — balandlik.

margin — tashqi bo‘shliq.

padding — ichki bo‘shliq.

.box {
  width: 200px;
  height: 120px;
  margin: 20px;
  padding: 15px;
  background-color: lightblue;
}
Kenglik: 200px, Balandlik: 120px

3️⃣ Fon va Ranglar

background-color — fon rangini belgilaydi.

background-image — fon rasmi qo‘yadi.

background-size — fon o‘lchamini belgilaydi.

background-repeat — fon takrorlanishini belgilaydi.

.bg-box {
  background-color: yellow;
  background-image: url('https://via.placeholder.com/100');
  background-size: cover;
  background-repeat: no-repeat;
}

4️⃣ Chegaralar va Yumaloqlash

border — chegara.

border-radius — yumaloqlash.

.rounded-box {
  border: 3px solid black;
  border-radius: 15px;
}

5️⃣ Effektlar

box-shadow — element soyasi.

opacity — shaffoflik.

transition — o‘tish effekti.

transform — o‘lcham/aylantirish/siljitish.

.effect-box {
  box-shadow: 0 4px 10px rgba(0,0,0,0.3);
  opacity: 0.9;
  transition: transform 0.3s;
}
.effect-box:hover {
  transform: scale(1.1);
}

6️⃣ Joylashuv (Layout)

display — element ko‘rinish turini belgilaydi (block, inline, flex, grid).

position — joylashuv turini belgilaydi (static, relative, absolute, fixed, sticky).

top / right / bottom / left — joylashuv koordinatalari.

z-index — element ustma-ust tartibini belgilaydi.

.pos-box {
  position: relative;
  top: 10px;
  left: 20px;
  z-index: 2;
}
Joylashuv: relative

7️⃣ Flexbox

display: flex — elementlarni moslashuvchan qator/sutunda joylashtiradi.

justify-content — asosiy o‘qda tekislash.

align-items — kesishgan o‘qda tekislash.

gap — elementlar orasidagi bo‘shliq.

.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}

8️⃣ CSS Grid

display: grid — elementlarni panjara tizimida joylashtiradi.

grid-template-columns — ustunlar soni va o‘lchami.

gap — kataklar orasidagi bo‘shliq.

place-items — gorizontal va vertikal tekislash.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;

list-style-type — ro‘yxat belgisi turi.

border-collapse — jadval chiziqlarini birlashtirish.

border-spacing — jadval kataklari orasidagi bo‘shliq.

ul {
  list-style-type: square;

🔟 Overflow va Scroll

overflow — ortiqcha kontent ko‘rinishini boshqaradi.

overflow-x / overflow-y — faqat gorizontal yoki vertikal.

scroll-behavior — skroll animatsiyasi.

.scroll-box {
  width: 150px;
  height: 80px;
  overflow: auto;
}
Bu matn juda uzun bo‘lib, qutidan chiqib ketadi, shuning uchun scroll paydo bo‘ladi.

1️⃣1️⃣ Cursor va Ko‘rsatkichlar

cursor — sichqoncha ko‘rsatkich turi.

pointer-events — element bosilishini boshqaradi.

user-select — matnni tanlash imkonini boshqaradi.

.cursor-box {
  cursor: pointer;
  user-select: none;
}
Menga bos, lekin matnni tanlab bo‘lmaydi.

1️⃣2️⃣ Animatsiya

@keyframes — animatsiya bosqichlarini belgilaydi.

animation-name — animatsiya nomi.

animation-duration — davomiyligi.

animation-iteration-count — necha marta takrorlanishi.

@keyframes move {
  0% { transform: translateX(0); }
  100% { transform: translateX(50px); }
}
.anim-box {
  animation: move 1s infinite alternate;
}

13️⃣ Transform — Aylantirish, Kattalashtirish, Siljitish

transform — element ko‘rinishini o‘zgartiradi.

rotate() — aylantirish.

scale() — kattalashtirish yoki kichiklashtirish.

translate() — joyini siljitish.

skew() — qiya qilish.

.transform-box {
  transform: rotate(10deg) scale(1.1) translate(10px, 5px);
}

14️⃣ Transition — O‘zgarishni Yumshoq Qilish

transition-property — qaysi property o‘zgaradi.

transition-duration — qancha vaqt davom etadi.

transition-timing-function — o‘zgarish tezligi.

.transition-box {
  background-color: orange;
  transition: background-color 0.5s, transform 0.5s;
}
.transition-box:hover {
  background-color: red;
  transform: scale(1.1);
}

15️⃣ Shadow Effektlar

box-shadow — elementga soya beradi.

text-shadow — matnga soya beradi.

.shadow-box {
  box-shadow: 4px 4px 10px rgba(0,0,0,0.3);
}
.shadow-text {
  text-shadow: 2px 2px 5px rgba(0,0,0,0.5);
}

Soyali matn

16️⃣ Fon Kengaytirilgan Usullar

background-size — fon rasmi o‘lchami.

background-position — fon rasmi joylashuvi.

linear-gradient — ranglar o‘tishi.

.bg-advanced {
  background-image: linear-gradient(to right, red, yellow);
}

17️⃣ Filter — Tasvir va Elementlarni O‘zgartirish

blur() — xiralashtirish.

brightness() — yorqinlik.

contrast() — kontrast.

grayscale() — qora-oq effekt.

sepia() — sariq effekt.

.filter-box {
  filter: grayscale(100%);
}

18️⃣ Responsive Dizayn

@media — ekran o‘lchamiga qarab style yozish.

min-width — minimal kenglik sharti.

max-width — maksimal kenglik sharti.

@media (max-width: 600px) {
  .responsive-box {
    background-color: lightgreen;
  }
}
Kichik ekranda yashil bo‘ladi

19️⃣ Pseudo-class va Pseudo-element

:hover — sichqoncha ustiga kelganda.

:active — bosilganda.

::before — elementdan oldin qo‘shimcha.

::after — elementdan keyin qo‘shimcha.

.button:hover {
  background-color: orange;
}

20️⃣ CSS Variables — O‘zgaruvchilar

--my-color — o‘zgaruvchi nomi.

var(--my-color) — o‘zgaruvchini chaqirish.

:root {
  --main-bg: pink;
}
.variable-box {
  background-color: var(--main-bg);
}
